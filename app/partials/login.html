By default, Gisto is using oAuth via github API. In order for you to get authenticated, you need to generate access token.
That is why GitHub username and password is asked by Gisto. After first authentication - only the generated token used to bridge between github and your gisto installation up to the point when you log out of gisto - action which terminates the token.

Other options of authentication are:
  • Manualy generated token
  • 2-step authentication (if 2 step authentication is active in your account)
